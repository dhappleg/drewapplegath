Chevy Altuve Challenge

2019 All-Star Baseball Game - Cleveland, OH

07/03/2019 - 07/10/2019

The Altuve Challenge is an interactive game where the user tries to hit the lighted up buttons as fast as possible. The challenge with this activation was arcade button lights that we used, since the light included from the manufacturer is single color, and not very bright. Instead I added a NeoPixel ring onto each button and developed a custom arduino hat to interface the buttons and the node server.
